# Mobile localization

The shared translation dictionary lives in:

- `mobile/localization/strings.en.json`
- `mobile/localization/strings.fr.json`

Generate platform resources with:

```
node mobile/localization/generate.mjs
```

This updates:

- iOS: `mobile/iosApp/iosApp/Resources/*/Localizable.strings`
- Android: `mobile/androidApp/src/main/res/values*/strings.xml`

Android resource keys are generated by replacing `.` with `_` in the shared keys.
